# GL-PROYECT

# Camara Movement
- Key: A
  - Moves camara in a cicle motion to the left
- Key: D
  - Moves camara in a circle motion to the right
- Key: S
  - Moves camara down
- Key: W
  - moves camara up
- Key: Q
  - Zoom in
- Key: E
  - Zoom Out

# Light Movement
- Key: Up arrow
  - Moves Light up
- Key: Down Arrow
  - Moves light down
- Key: Left Arrow
  - Moves light to the left
- Key: Right Arrow
  - Moves camara to the right
